How I play
the game.

A small set of working principles I keep unlocking, project after project. Less hype, more craft.

01
No build step

Plain HTML/CSS/JS. Fewer moving parts, less to break in 2030.

02
Docs first

README · CHANGELOG · ROADMAP. If a feature isn't documented, it isn't done.

03
Privacy by default

No third-party scripts. Privacy is the design, not a setting.

04
Capability detection

Check the system, don't assume it. Hide what isn't supported.

05
Cloudflare edge

Workers + tunnels, not VPS sprawl. Predictable, cheap, fast.

06
Calm motion

Felt, not noticed. Reduced-motion always honored, everywhere.
